# APP DEV 2 - Placement Portal Application
The application is built as a decoupled Single Page Application. 
The backend is a modular Flask REST API with blueprint routing, SQLite database management, and SQLAlchemy ORM. 
The frontend is built with Vue.js 3, utilizing client side routing and Bootstrap 5 CDN for a responsive user interface. 
The system includes role based authentication, application tracking, and resume file upload features
